08 - Material In-Out Transfer Report

Description: Track materials moving in and out across JD locations.

Formatting Rules:

  • FROM / TO Column: Must use format: JD Cityname (e.g., JD Chennai, JD Bangalore LABS).
    ❌ Incorrect examples: JUSTDIAL,KOLKATTA, Justdial LTD(Noida)
    ✅ Correct: JD Hyderabad, JD Mumbai
  • SENDER / RECEIVER Column: Must include person’s name – not location.
    ❌ Don’t enter: JD Pune
    ✅ Correct: Ramesh Kumar, Sneha J
  • Asset Type Column: Use only generic names, not models or counts.
    ❌ Don’t enter: Dell 390 MBD, 5 Mouse, 3 MBD
    ✅ Correct entries: Motherboard, Printer Cartridge, Complete PC, Keyboard, Network Switch.
    Use Remarks column for Make, Model, Quantity.
  • Serial Number / Barcode: Mandatory for all JD assets.
    ❌ Don’t leave blank or NA
    ✅ For non-serial items, use NR (Not Required).
  • Gate Pass / Delivery Challan: Outward materials require Gate Pass; Inward materials require Delivery Challan No.
  • Taiga / IT Request Number: Required for all outward materials; no blanks allowed.
  • IT Approval: Mandatory for all outward materials.
